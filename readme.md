**Audio Deepfake Detector** is a specialized tool designed to verify the authenticity of vocal recordings. By analyzing acoustic patterns and spectral features, it can identify whether an audio file is a genuine human recording or a synthetic voice generated by AI (Deepfake).

This project is an implementation of my undergraduate thesis research at **Darussalam Gontor University**, focusing on **Semi-Supervised Contrastive Learning** for audio feature representation.

## ‚ú® Key Features
- **Acoustic Feature Extraction**: Uses `librosa` to extract Mel-Frequency Cepstral Coefficients (MFCC).
- **Deep Learning Inference**: Integrated with PyTorch for real-time classification.
- **CLI Interface**: Easy-to-use terminal-based tool for quick analysis.
